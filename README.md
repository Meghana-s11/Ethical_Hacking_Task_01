# Ethical_Hacking_Task_01  Information Gathering & Reconnaissance

## 📌 Overview

This repository contains the submission for **Ethical Hacking Task 01**, which focuses on the **Information Gathering (Reconnaissance)** phase of the ethical hacking methodology.

The objective of this task was to perform **passive reconnaissance** on a publicly accessible website using open-source intelligence (OSINT) techniques and publicly available tools. All information collected was obtained ethically without interacting with or attempting to exploit the target system.

---

## 🎯 Objective

The primary objectives of this task were to:

- Understand the importance of the reconnaissance phase in ethical hacking.
- Perform passive information gathering using publicly available resources.
- Identify domain registration details using WHOIS.
- Analyze DNS records of the target domain.
- Identify web technologies used by the target website.
- Evaluate HTTP Security Headers.
- Examine publicly accessible configuration files such as `robots.txt` and `sitemap.xml`.
- Document all findings in a structured reconnaissance report.

## 🔍 Reconnaissance Activities Performed

### 1. Target Selection
- Selected a publicly accessible educational website.
- Verified website accessibility.

### 2. WHOIS Lookup
- Collected domain registration details.
- Identified registration dates and domain status.
- Reviewed DNS name servers.

### 3. DNS Enumeration
- Examined A Records.
- Examined MX Records.
- Examined NS Records.
- Examined TXT Records.

### 4. Website Technology Identification
- Identified technologies and hosting infrastructure.
- Reviewed website architecture using public technology detection tools.

### 5. HTTP Security Header Analysis
- Evaluated security headers.
- Identified missing security headers.
- Reviewed overall security rating.

### 6. Public Configuration Files
- Examined the availability of:
  - robots.txt
  - sitemap.xml

---

## 📊 Key Findings

The reconnaissance exercise successfully collected publicly available information regarding the target website's infrastructure.

Key observations include:

- Public WHOIS information was available through ICANN Lookup.
- DNS records were successfully identified.
- The website utilizes modern web technologies.
- Multiple HTTP security headers were missing during analysis.
- Public configuration files were reviewed for indexing information.
